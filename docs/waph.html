Web Application Programming and Hacking

This course covers the fundamentals of web application development and security, including front-end and back-end programming, client-server communication, and common web vulnerabilities such as cross-site scripting (XSS). Through a series of labs, hackathons, and individual projects, students gain hands-on experience building web applications and learning how to identify and defend against real-world attacks.
